## 三、关键决策与铁律

详见 [`ARCHITECTURE.md`](./ARCHITECTURE.md)。核心:

1. **单文件 HTML**:不引框架,所有 CSS/SVG 内联,可独立运行
2. **统一苹方字体**(v3.2 后):靠字重 + 字号 + 字距建立层级,不用衬线/等宽
3. **8px 垂直节奏**(v3.1 spacing pass):section 112px / h3 72px / callout 36px
4. **章节色系铁律**:10 个 `--c-XXX` 变量,绝对不在元素 `style` 里硬编码颜色
5. **AI 图 prompt 禁词**:hex code / ALL CAPS / "NO TEXT" / "book/magazine" 都会触发文字串扰
6. **文献引用强制**:每个尖锐结论必须有 `(Author Year)` 标注

## 四、工具链 · generation/

### 脚本(`generation/scripts/`)

| 文件 | 功能 |
|---|---|
| `img_pipeline.py` | 通用图片下载 + 压缩(sips)CLI 工具 |
| `embed_heroes.py` | 把 10 张 hero 嵌入 HTML 对应章节 + 添加 CSS |
| `integrate_v2.py` | v2 集成脚本(13 个章节 fragment 合并 + 视觉添加) |
| `integrate_v3.py` | v3 集成(13 个视觉组件批量插入) |
| `integrate_v4.py` | v4 集成(spacing + 字体 pass) |
| `charts.py` | matplotlib 绘制 RHR/HRV/周量趋势图(基于 Coros 6 个月数据) |

### Prompts(`generation/prompts/`)

| 文件 | 说明 |
|---|---|
| `hero_v1.py` | 初版 prompt(含 hex code、ALL CAPS、否定指令 → ❌ 触发文字串扰) |
| `hero_v2.py` | 去 hex + ALL CAPS,但仍有 "editorial illustration for a book" → ⚠️ §9/§10 出现假书页 |
| `hero_v3.py` | 最终版,纯描述性自然语言,"single cinematic frame, not a book" → ✅ 全部干净 |

**推荐**:重新生成图时一律用 v3,v1/v2 留作教训档案。
